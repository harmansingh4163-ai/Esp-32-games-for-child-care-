# ESP32-S3 Game Center

A multiplayer touchscreen game device built on the ESP32-S3, designed as a 
low-cost alternative to commercial activity tables for childcare centers. 
9+ games including Chess, Ludo, Connect Four, Snakes & Ladders, and Dots & 
Boxes, with a 3×3 grid menu. Rendering via LovyanGFX (migrated from LVGL 
for color-pipeline and PSRAM reasons).

Status: active development.

**12-game touch arcade for ESP32-S3 (800x480, GT911) — Chess, Ludo, Match Shapes & more. 2-player local multiplayer. Built with Arduino IDE + LovyanGFX. Requires PSRAM.**
